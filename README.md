# CRM Application for Banquet Hall Booking

## Overview
This project is a **Salesforce-based CRM application** designed to manage and streamline the banquet hall booking process. It helps handle customer inquiries, booking confirmations, event planning, and payment tracking efficiently.

The system ensures smooth coordination between clients and banquet management, reducing manual effort and avoiding booking conflicts.

## Objectives
- Automate banquet hall booking workflow
- Manage customer inquiries and event details
- Prevent double booking of halls
- Track payments and generate reports
- Improve customer experience through automation

## Features

### Customer Management
- Store customer details (Name, Phone, Email)
- Maintain customer history

### Inquiry Management
- Capture event requirements
- Track inquiry status (New, In Progress, Converted)

### Banquet Hall Management
- Manage hall details (Capacity, Pricing, Availability)

### Booking Management
- Confirm and manage bookings
- Avoid scheduling conflicts

### Event Planning
- Manage catering, decoration, and arrangements

### Payment Tracking
- Track advance and pending payments
- Monitor payment status

## Technologies Used
- Salesforce CRM
- Apex
- Flow Builder
- Data Modeling
- Validation Rules
- Reports & Dashboards

## Automation Implemented
- Auto-create booking from approved inquiry
- Email notifications for booking confirmation
- Reminder notifications before event
- Validation to prevent double booking
